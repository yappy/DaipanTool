DaipanTool
非想天則オフライン対戦自動記録ツール

*ソース等も入っているセットをダウンロードした方へ
distディレクトリがビルド済み配布パッケージになります。
Java7でビルドされているので注意。実行にはJRE7が必要です。

*起動方法
daipan.jarをダブルクリックするとか。
コマンドライン上での起動方法は次の通り。
※%はコマンドを表す。%自体はコマンドに書いてはいけない。
% java -jar daipan.jar

*注意
CPU対戦やリプレイの再生も自動的に記録してしまいます。
×を押しても終了しません。隠れるだけです。
「終了」ボタンやアイコン右クリックメニューから終了させてください。

*使い方
非想天則と同時に動かしてください。
非想天則を発見すると"find."のメッセージがログに出ます。
対戦の決着がつくと、(日時).csvのファイルに結果が追記されます。

*記録例
2011-09-05 06:55:24,yappy,アリス,2,にとり使いたくない,萃香,1,,1315173324852,3,9
日時,1Pプロファイル名,1Pキャラ名,1P取得ラウンド数,2Pプロファイル名,2Pキャラ名,2P取得ラウンド数
となっています。取得ラウンド数が2の方が勝者です。
一応引き分けもあり得たような気がしたのでこうなっています。
右の方の数値は集計時に便利かもしれない生データです。
1970/1/1_0:00からのms単位の日時,1PキャラID,2PキャラID
となっています。
とりあえずExcelで処理するのが簡単かなと思ったのでこの形式にしました。
要望があれば言ってください。

*ビルド
build.xmlがAntのビルドファイルです。コマンドでの起動は以下の通り。
% ant
デフォルト以外のターゲットについてはbuild.xml参照。以下例。
% ant clean
EclipseにはAntが同梱されており、特別なインストール作業なしに利用できます。
パッケージエクスプローラからbuild.xmlを右クリックしてRun As->Ant build。
上のツールバーの右矢印の実行ボタンでもOK。

srcに.javaソースおよびjarに含めるリソースがあります。
binにsrcからソースをコンパイルした.classファイルが出力され、リソースはコピーされます。
binの内容がjarファイルにまとめられます。
distに配布パッケージの内容がコピーされます。
